1. Orientado a la Web (pero multiuso)

Javascript es el lenguaje de programación principal para el desarrollo web del lado del cliente. Esto significa que se ejecuta en los navegadores web y permite la creación de páginas web interactivas. Sin embargo, aunque originalmente estuviera diseñado para funcionar en navegadores,hoy en día también es posible utilizar Javascript en aplicaciones móviles y aplicaciones de escritorio.

2. Lenguaje interpretado y dinámico

A diferencia de muchos otros lenguajes de programación, Javascript es interpretado, no compilado. Esto significa que puedes escribir el código y verlo ejecutarse en tiempo real en tu navegador sin necesidad de un paso intermedio de compilación

3. Compatibilidad y estandarización

JavaScript es estandarizado por ECMA International como ECMAScript. Esta estandarización garantiza que las implementaciones de JavaScript sean consistentes a través de diferentes navegadores y plataformas. Las versiones de ECMAScript (como ES5, ES6, etc.) introducen mejoras y nuevas características al lenguaje de manera regular.

4. Asincronía y manejo de eventos

JavaScript es conocido por su capacidad para manejar operaciones asincrónicas mediante el uso de callbacks, promesas y la sintaxis async/await introducida en ES8. Esta característica es crucial para operaciones que toman tiempo, como solicitudes de red o temporizadores, permitiendo que el código siga ejecutándose mientras se espera una respuesta. Esta capacidad de manejo de eventos y asincronía es fundamental para crear aplicaciones web reactivas.

5. Programación orientada a objetos

A diferencia de lenguajes orientados a objetos tradicionales como Java o C++, que utilizan clases para definir estructuras y comportamientos de los objetos, JavaScript utiliza un modelo basado en prototipos. En este modelo, los objetos pueden heredar propiedades y métodos directamente de otros objetos

6. Ecosistema y herramientas

El ecosistema de JavaScript es uno de los más grandes y activos, con una amplia variedad de bibliotecas y frameworks como React, Angular y Vue.js, que facilitan el desarrollo de aplicaciones web complejas. Además, Node.js ha ampliado el uso de JavaScript al lado del servidor, permitiendo el desarrollo de aplicaciones de backend con el mismo lenguaje utilizado en el frontend.

A continuación se listan los principales tipos de datos en Javascript:

1. Number

Representa tanto números enteros como decimales. Ejemplos:

 
                    number1 = 42 // Number Integer
                    number2 = 3.1415 // Number Float

2. String

Una cadena de caracteres, utilizada para representar texto. Ejemplos:


                    string1= "Hola Mundo"
                    string2= "Iñigo"

3. Boolean

Representa un valor lógico, puede ser true o false. Ejemplos:

                    
                    1==0 //false
                    1==0 //true

4. Null

Representa la ausencia intencional de cualquier valor. Es un valor único que significa "vacío" o "nulo". Ejemplo:


                    const vacio= null

5. Undefined

Indica que una variable ha sido declarada pero no se le ha asignado un valor.Ejemplo:


                    let variable
                    console.log(variable)// devolverá undefined

6. Object

Colección de propiedades, donde cada propiedad es una asociación entre un nombre (clave) y un valor. Ejemplo:


                    const muro={
                        tipoMuro: "trasdosado de pladur",
                        longitudMuro: 3,
                        areaNetaMuro: 4.235,
                        esEnvolventeTermica: false,
                    }

`toUpperCase`

Convierte todos los caracteres de la cadena a mayúsculas. Ejemplo:

                    const holaMundo = "Hola, mundo";
                    console.log(holaMundo.toUpperCase()); // "HOLA, MUNDO"

`charAt(index)`

Devuelve el carácter en la posición especificada dentro de la cadena. Ejemplo:

                    const hola = "Hola";
                    console.log(hola.charAt(1)); // Devuelve el caracter en la posición 1 del string, en este caso "o"

`substring(start, end)`

Devuelve una parte de la cadena entre los índices start y end (sin incluir end). Ejemplo:

                    let holaMundo = "Hola, mundo";
                    console.log(holaMundo.substring(0, 3)); // "Hol"

Un condicional es una estructura de control que te permite ejecutar diferentes bloques de código en función de una condición. La forma más común de un condicional en Javascript es la instrucción if-else:


                let puntuacion = 4.8;
                if (puntuacion >= 5) {
                console.log("Has aprobado el examen");
                } else {
                console.log("Lo siento, has suspendido el examen");
                }

En este ejemplo, se comprueba si la variable puntuacion es mayor o igual que 5. En el caso de que esta condición sea verdadera, imprime en consola "Has aprobado el examen En el caso de que la condición sea falsa, se ejecuta el siguiente bloque, imprimiendo en consola "Lo siento, has suspendido el examen"

El operador ternario es una forma compacta de escribir una sentencia if-else. A dierencia de las estructuras de control if-else, solamente comprueba una condición. Tiene tres partes: una condición, una expresión a ejecutar si la condición es verdadera, y otra expresión a ejecutar si la condición es falsa. Su sintaxis es: resultado = condicion ? valorVerdadero : valorFalso 


                let edad = 20;
                let mensaje = edad >= 18 ? "Eres mayor de edad." : "Eres menor de edad.";
                console.log(mensaje); // "Eres mayor de edad."

En este caso, se comprueba la variable edad, si esta es mayor que 18, se devuelve "Eres mayor de edad" en el caso de que esta condición sea falsa devuelve "Eres menor de edad"

Declaración de Función

Una declaración de función define una función con un nombre y la palabra clave function. Esta función se puede llamar en cualquier parte del código, incluso antes de que la función sea definida, debido a un proceso llamado hoisting.

                
                function multiplicar(a, b) {
                return a * b;
                }
                console.log(multiplicar(3, 4)); // 12

Expresión de Función

Una expresión de función crea una función que se puede almacenar en una variable. Esta función puede ser anónima o tener un nombre. A diferencia de la declaración de función, la expresión de función no es hoisted, lo que significa que no se puede llamar antes de su definición.

                const saludar = function() {
                    console.log("Hola, mundo!");
                };
                
                saludar(); // Llama a la función, imprime "Hola, mundo!"

La palabra clave this en Javascript es una referencia a un objeto, y su valor depende del contexto en el que se usa. Vamos a ver algunos de los contextos más comunes:

Contexto Global

En el contexto global (fuera de cualquier función o bloque de código), this se refiere al objeto global. En un navegador web, este objeto es window.


                console.log(this); // En un navegador, imprime el objeto window

En el Contexto de un Objeto

Cuando this se usa dentro de un método de un objeto, se refiere a ese objeto específico.


                const individuo = {
                    nombre: "Iñigo",
                    saludar: function() {
                        console.log(`Hola, soy ${this.nombre}`);
                    }
                };
                
                individuo.saludar(); // Imprime "Hola, soy Iñigo"

En este caso, se define una función como parte del objeto "individuo". Al llamar a la función es posible utilizar la palabra "this" para acceder a alguna de las keys del objeto
